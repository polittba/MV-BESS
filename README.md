# MV-BESS

Mountain View Battery Energy Storage System (MV BESS)— IT/Control System Architecture documentation.
This knowledge base is intended to create the IT Business and System Requirements to support the MV BESS Project

## Structure

- `61850/` — IEC 61850 reference models for ESS
- `docs-AssetManagement/` — Asset Performance Management documentation for ESS
- `docs-BESSCore/` — BESS Core documentation (component hierarchy, alarm philosophy)
- `docs-MiscRef/` — Miscellaneous reference material (architecture vision, MODBUS, vendor tiers, media assets)
- `docs-Word/` — Source Word documents
